# app store in react-native

## used library

- Redux.
- reactnavigation
- styled-components

### Features

- Lista de produtos.
- Carrinho de Compra.

### Install

- android
  `yarn install`
  `yarn run android`

- ios
  `yarn install`
  `npx pod-install ios`
  `yarn run ios`

### Tests

`yarn test`

- For see the coverage
  `yarn test --coverage`
